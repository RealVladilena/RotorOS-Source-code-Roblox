# RotorOS - Système Avancé d'Éoliennes pour Roblox 🌬️

Ce dépôt GitHub contient uniquement les **scripts Lua** à copier dans Roblox Studio pour faire fonctionner votre système d’éolienne, incluant le contrôle RPM, la logique YAW/Pitch et le système SCADA. L'explication complète et détaillée du projet est disponible sur la page de présentation : https://rotoros.prts-vision.com/project/.

## 📝 À propos du projet
Créé par un technicien éolien passionné de développement, le projet RotorOS vise à repousser les limites de la simulation sur Roblox. Ce framework Lua est conçu pour être universel, réaliste et s'adapter facilement à différents constructeurs d'éoliennes. 

## 🛠️ Installation et Utilisation
1. Ouvrez Roblox Studio et chargez votre place.
2. Ajoutez les scripts dans l’endroit prévu en vous référant à la documentation.
3. Vérifiez et adaptez la hiérarchie de vos objets 3D (Rotor, pales, nacelle, pivot YAW, etc.).
4. Lancez le jeu et ajustez les paramètres physiques (cut-in/cut-out, RPM, limites) pour correspondre à votre turbine.

## 🌐 Documentation
Absolument tout est expliqué dans notre documentation en ligne, qui centralise les étapes de configuration et les tutoriels d’adaptation. Vous y retrouverez également tous les scripts en copie rapide directement sur : https://rotoros.prts-vision.com/docs/.

## 🚀 Objectif
L'objectif est de fournir un framework de simulation éolienne performant sur Roblox. Cette base de type système d'exploitation (RotorOS) permet une gestion avancée et adaptable à de multiples modèles.

## 🤝 Contribution et Support
Ce projet constitue une base de travail ouverte à la communauté des développeurs Roblox. N'hésitez pas à explorer le code, à suggérer des améliorations ou à adapter les variables pour créer vos propres éoliennes interactives.
